# Task 008: Автозаполнение ID доски при вводе URL

## Цель

В блоке настроек связи с другой доской (режим "Ссылка") добавить возможность ввода полного URL доски Jira. При вводе
корректного URL поле "ID связанной доски" должно автоматически заполняться извлеченным идентификатором.

## Технические изменения

### 1. `src/utils/JiraUtils.ts`

Необходимо модифицировать функцию `getBoardIdFromUrl`, чтобы она могла принимать строку с URL в качестве аргумента.

- Изменить сигнатуру: `export const getBoardIdFromUrl = (urlStr?: string): string | null => ...`
- Если `urlStr` передан, использовать его для парсинга. Если нет — использовать `window.location`.
- Расширить логику парсинга, чтобы поддерживать следующие паттерны:
    - Параметр `rapidView` в query params (текущая реализация): `.../RapidBoard.jspa?rapidView=123`.
    - Идентификатор в пути URL (REST-style): `.../boards/123`.

### 2. `src/pages/Settings/Settings.tsx`

В секции `boardMode === 'link'` добавить поле для ввода URL доски.

- Добавить новое поле ввода `<input type="text" />` для URL.
- Реализовать обработчик `onChange` для этого поля:
    1. Вызывать обновленную `getBoardIdFromUrl(inputValue)`.
    2. Если ID успешно извлечен, вызывать `setLinkedBoardId(id)`.
- Поле "ID связанной доски" должно оставаться доступным для ручного ввода и редактирования, чтобы пользователь мог
  внести правки, если автоматическое определение сработало неточно.

## План реализации

1.  [ ] Обновить функцию `getBoardIdFromUrl` в `src/utils/JiraUtils.ts`, добавив поддержку аргумента и расширив
    регулярные выражения/логику парсинга.
2.  [ ] Импортировать `getBoardIdFromUrl` в `src/pages/Settings/Settings.tsx`.
3.  [ ] Добавить в JSX компонента `Settings` блок с новым полем ввода URL в режиме `link`.
4.  [ ] Реализовать синхронизацию между полем URL и состоянием `linkedBoardId`.
5.  [ ] Провести тестирование на различных форматах ссылок Jira.

## Критерии приемки

- [ ] При вводе ссылки с `rapidView=XXX` поле ID автоматически принимает значение `XXX`.
- [ ] При вводе ссылки с `/boards/XXX` поле ID автоматически принимает значение `XXX`.
- [ ] Пользователь может вручную изменить ID после того, как он был подставлен из URL.
- [ ] Сохранение настроек работает корректно (сохраняется именно `linkedBoardId`).
