# Задание: Реализация возможности привязки доски (Link Mode)

## Контекст

Сейчас проект поддерживает только хранение данных непосредственно на текущей доске (`type: 'data'`). Необходимо добавить
функционал, позволяющий одной доске ссылаться на другую доску через её `boardId`.

## Что уже сделано (частично)

- В `src/types/JiraRiceFarmTypes.ts` тип `JRFBoardData` уже является Union-типом:
  `JRFBoardDataWithIssues | JRFBoardDataLink`.
- Тип `JRFBoardDataLink` содержит `type: 'link'` и `linkedBoardId: string`.
- Заведена функция сохранения данных типа `link` в `JiraBoardDataStore.modifyBoardLinkAndSave`

## Что нужно доделать

### 1. Обновление типов и состояния в `src/pages/Settings/Settings.tsx`

- Изменить тип состояния `formData` с `JRFOnlyBoardData` на `JRFBoardData`.
- Реализовать переключение режимов в UI: "Данные" (Data) и "Ссылка" (Link).
- При выборе режима "Ссылка":
    - Отображать текстовое поле для ввода `linkedBoardId`.
    - Скрывать все остальные поля настроек (Reach, Impact, Confidences и т.д.).
- При выборе режима "Данные":
    - Отображать стандартный набор полей (как сейчас).
    - Если текущий режим был "Ссылка", при переключении на "Данные" необходимо инициализировать `formData` значениями по
      умолчанию (`initFormData()`).

### 2. Валидация

- Обновить функцию `validateFormData` в `src/pages/Settings/Settings.tsx`, чтобы она принимала `JRFBoardData`.
- Добавить проверку: если выбран режим "Ссылка", поле `linkedBoardId` не должно быть пустым.
- Если выбран режим "Данные", использовать существующую логику валидации для `JRFOnlyBoardData`.

### 3. Загрузка данных

- Убедиться, что при открытии настроек `useEffect` корректно загружает данные любого типа (`data` или `link`) и
  устанавливает соответствующий режим в UI.

## Критерии успеха

1. В настройках появился переключатель "Данные / Ссылка".
2. Можно переключиться на "Ссылка", ввести ID другой доски и сохранить.
3. После сохранения при открытии настроек отображается режим "Ссылка" и введенный ID.
4. Можно переключиться обратно на "Данные", после чего отображаются стандартные поля.
5. При попытке сохранить пустой `linkedBoardId` в режиме ссылки, срабатывает валидация.
